## Syntax

   ```bash
   python3 fileorganizer_linux.py -s <SOURCE_FOLDER> -o <OUTPUT_FOLDER>
   ```

   - If you want to delete sub folders:
   ```bash
   python3 fileorganizer_linux.py -s <SOURCE_FOLDER> -o <OUTPUT_FOLDER> -d
   ```
